subroutine math/randn(x)¶

Populates the array x with normally-distributed random variates with zero mean and unit variance using the polar form of the Box-Müller transform. The length of x must be even otherwise the subroutine will stop.

Parameters:x (*) [real,out] :: was 2d0 to trigger do while

Module core¶

Description

The core module contains the main subroutines that AADG3 uses to generate the timeseries for an oscillation mode.

Quick access

Needed modules

subroutine core/generate_kicks(n_fine, cadence, tau, nsd, kicks)¶

Generates an exponentially-damped random walk (i.e. an AR(1) process) that AADG3 uses to model granulation, for either the correlated or uncorrelated component of the driving term.

Parameters:
  • n_fine [integer,in]
  • cadence [real,in]
  • tau [real,in]
  • nsd [real,in]
  • kicks (*) [real,out]
